Configure seu Firebase

Acesse o Console do Firebase, abra seu projeto → ⚙️ Configurações → Seus apps → ícone </> → copie o objeto firebaseConfig e cole abaixo.

Passo a passo:
1. console.firebase.google.com
2. Seu projeto → ⚙️ Configurações do projeto
3. Role até "Seus apps" → clique em </> (Web)
4. Registre o app → copie o bloco firebaseConfig = { … }
5. Cole aqui embaixo e clique em Salvar
